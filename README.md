# Vegan Home Cooks

A community recipe book for the [Vegan Home Cooks](https://veganhomecooks.online) 
community. Find us also on [Discord](https://discord.gg/DBXGfxweAr), 
[Reddit](https://reddit.com/r/veganhomecooks), and 
[Lemmy](https://lemmy.world/c/veganhomecooks).

## How it's built

This site is built with [Hugo](https://gohugo.io), a static site generator. 
The source is hosted on GitHub and deployed automatically via 
[Cloudflare Pages](https://pages.cloudflare.com).

## Theme

This site uses the [hugo-cookbook](https://github.com/deranjer/hugo-cookbook) 
theme by [deranjer](https://github.com/deranjer).
